Qu’est-ce que eBay Sidebar pour Firefox?

eBay Sidebar pour Firefox est un complément pour le navigateur Mozilla Firefox, qui vous permet d’assurer le suivi de vos transactions d’achat et de vente sur eBay. Il vous propose une vue d’ensemble de Mon eBay ainsi que des alertes personnalisables dans le navigateur relatives aux événements eBay, par exemple lorsque quelqu’un a surenchéri sur vous ou que vous avez vendu ou acheté un objet.
